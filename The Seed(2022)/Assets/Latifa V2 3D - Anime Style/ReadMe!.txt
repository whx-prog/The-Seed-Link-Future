This asset contain character with few animations(no root motion) and several facial emotions and expressions
such as root motion. The character is fully rigged as humanoid. The clothing and hairs are also rigged.
This character also comes with several blendshapes which you can use to create your own emotions and animations.

***Please rate and review if you like this character***

INFO:

Character Rig: Humanoid 
Cloth Rig: Yes
Hair Rig: Yes

Animations: Walk, Run, Idle, Bow. 
More will be added in future update. Animaitons with root motion will also be added in future updates.

Mixamo Animations: Supports all mixamo animaions.

Facial Expressions: Blink, Angry, Fun, Joy, Surprised, Sorrow.

Blend Shapes: Many
You can use these blend shapes to create custom emotions. To use the blend shapes:
Go to character heirachy> body > Skinned mesh renderer > click blend shape drop down.
You will find all the blend shapes listed there.
